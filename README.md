# Property Rental and Booking Platform

# Description

Property Rental and Booking Platform is a full-stack web application that allows users to browse, list, and book rental properties. The platform enables property owners to add and manage listings, while users can explore available properties, view details, and make bookings seamlessly.

This project demonstrates the implementation of modern web technologies to create a scalable and user-friendly property rental system.

# Features

- User Authentication (Signup/Login/Logout)
- Browse Property Listings
- View Property Details
- Add New Property Listings
- Edit and Delete Listings
- Image Upload for Properties
- Responsive User Interface
- Secure Data Storage
- Flash Messages and Error Handling


# Tech Stack

Frontend: HTML, CSS, JavaScript,Bootstrap
Backend: Node.js ,Express.js
Database: MongoDB ,Mongoose
Other Tools & Services: Cloudinary (for image upload) ,Map Integration (Location display) ,Git & GitHub (Version Control)


# Installation & Setup

Follow these steps to run the project locally:

1. Clone the repository

git clone https://github.com/your-username/property-rental-platform.git

2. Navigate to the project folder

cd property-rental-platform

3. Install dependencies

npm install

4. Setup Environment Variables

Create a `.env` file and add:

CLOUDINARY_CLOUD_NAME=your_cloud_name  
CLOUDINARY_KEY=your_key  
CLOUDINARY_SECRET=your_secret  
MAP_API_KEY=your_map_api_key  

5. Start the server

node app.js

6. Open in browser

http://localhost:8080
